# 기여 가이드

## 원칙 수정/추가하기

1. `principles.md` 파일을 열어 원칙을 수정하거나 추가합니다.
2. Pull Request를 올립니다.
3. 크루들의 동의를 얻으면 머지됩니다.

GitHub 웹 에디터에서도 바로 수정할 수 있어요 → `principles.md` 파일 열기 → 연필 아이콘 클릭

## 원칙 파일 형식

```markdown
# 제목 (H1)

부제목
여러 줄도 됩니다.

1. 원칙은 이렇게
2. 번호 목록으로

> * 푸터 문장
```

## 디자인/기능 개선

이슈를 먼저 등록하고 PR을 올려주세요.
